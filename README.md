# Biliup

bilibili投稿客户端，理论上支持Windows，Linux，macOS。

主要是为了解决现有网页端不能多p投稿的问题，虽然现有b站客户端可以多p
但是有几个问题：
* 不支持多平台
* 投稿线路对国外不友好，速度较慢稳定性较差
* 不能批量选择文件，多p只能一个一个文件选择

[下载地址](https://github.com/ForgQi/Caution/releases/download/v0.0.1/biliup.exe)
